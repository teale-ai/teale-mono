## What Was Built

### WalletKit Module (New)
Added Solana USDC integration as a new `WalletKit` module bridging on-chain USDC to the existing CreditKit credit ledger.

- **Peg**: 1 credit = $0.0001 USD (10,000 credits = $1 USDC)
- `SolanaIdentity`: Ed25519 keypair in Keychain (CryptoKit Curve25519, same as WANKit pattern)
- `SolanaRPCService`: Actor wrapping solana-swift for USDC balance queries
- `DepositMonitor`: Polls Solana RPC every 30s for incoming USDC deposits
- `WithdrawalService`: Sends USDC via SPL token transfers
- `WalletBridge`: @Observable class bridging on-chain state to CreditKit (optimistic debit + compensating credit on failure)
- `CreditAmount+USD`: Extension for usdValue, usdFormatted, fromMicroUSDC()
- `Base58`: Encode/decode for Solana addresses
- New dependency: `p2p-org/solana-swift` (5.0.0+)
- CreditKit updated with `.deposit`/`.withdrawal` transaction types and `txSignature` field
- UI: Settings toggle for Solana wallet (devnet/mainnet), WalletView deposit/withdraw section, USD equivalents
- All localized in 5 languages

### Keep Mac Awake Setting
Added `IOPMAssertionCreateWithName` with `kIOPMAssertPreventUserIdleSystemSleep` toggle in Settings > General. Screen can still turn off; only system sleep is inhibited. Persists via UserDefaults.

### WAN Toggle Fix
WANManager.enable() was throwing on relay.connect() (DNS failure for the not-yet-deployed relay server), causing the toggle to silently bounce back to off. Fixed by making relay connect and discovery registration non-fatal — node enters "WAN enabled, relay unavailable" state and retries.

### Website Network Stats
Added a network stats bar to teale.com (teale-www repo) showing Devices Online, Total Network RAM, and Tokens Served. Fetches from a `stats.json` file that can be updated periodically.

## Key Decisions

- **Solana over Base/Ethereum**: Cheaper fees ($0.00025/tx), faster finality (0.4s), same Ed25519 crypto as WANKit, better Swift SDK
- **No Stripe/tradfi**: User explicitly wants strictly digital wallets
- **Credit peg rationale**: Maps existing pricing formula to market rates (~50-70% of cloud API pricing). Welcome bonus of 100 credits = $0.01
- **WAN should be resilient**: Most users will use WAN to contribute idle compute, not consume inference. The toggle must work even before the relay server is deployed.
