# Getting Started

Everything you need to install Teale and start using it, whether you want to chat with AI models, build applications, or earn by sharing compute.

---

## Install

Choose your platform to get Teale running.

| Platform | Guide | Method |
|----------|-------|--------|
| **macOS** (GUI + CLI) | [Install on Mac](install-mac.md) | DMG or `brew install --cask teale` |
| **macOS / Linux** (CLI only) | [Install the CLI](install-cli.md) | `brew install teale` or binary download |
| **Linux / Windows / Android** | [Cross-platform install](install-cross-platform.md) | `cargo install teale-node` or Docker |

---

## Quickstarts

Pick the path that matches what you want to do.

### Use Teale

Run AI models locally on your Mac. No accounts, no API keys, no cloud. Install the app, and a model downloads automatically based on your hardware.

[Quickstart: Chat](quickstart-chat.md)

### Build on Teale

Teale exposes an OpenAI-compatible API at `localhost:11435`. If your code works with OpenAI, it works with Teale --- just change the base URL.

[Quickstart: API](quickstart-api.md)

### Earn with Teale

Share your idle compute with the Teale network and earn USDC. Teale auto-selects the best model for your hardware and handles everything else.

[Quickstart: Earn](quickstart-earn.md)
