# teale login

Sign in to your Teale account via phone OTP.

## Synopsis

```
teale login
```

## Description

Starts an interactive sign-in flow. You will be prompted to enter your phone number, then a one-time verification code sent via SMS.

This command is interactive and cannot be used in non-interactive scripts.

## Options

None.

## Examples

```bash
teale login
```
