# Teale

Teale ships today as desktop apps for macOS and Windows. These docs only cover the functionality that is in the currently released apps.

---

## What the released apps do

- Chat locally with a loaded model from the Home tab.
- Route demand to live Teale Network models with transparent prompt and completion pricing.
- Supply inference from the same app and earn availability credits plus served-inference revenue.
- Manage a device wallet, export the ledger as CSV, and send Teale credits to device IDs or account identifiers.
- Sign in, link devices to a human account, and inspect linked devices from the Account tab.
- Switch the UI between English, Spanish, Portuguese (Brazil), and Filipino (Philippines), and toggle display units between Credits and USD.
